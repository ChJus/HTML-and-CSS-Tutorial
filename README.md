# HTML-and-CSS-Tutorial

Tutorial on HTML and CSS inspired by [Jeremy Thomas' Web Design in 4 minutes](https://github.com/jgthms/web-design-in-4-minutes) and [CSS Zen Garden](https://csszengarden.com/).

In this repository you can find:
- [Source code](https://github.com/ChJus/HTML-and-CSS-Tutorial/tree/main/docs) for creating styled versions of the tutorial website.

Steps:
1. Start by accessing the [website](https://chjus.github.io/HTML-and-CSS-Tutorial/).
2. Go through all of the sections and come back here to start experimenting.
3. [Download](https://github.com/ChJus/HTML-and-CSS-Tutorial/archive/main.zip) the repository.
4. Start styling the webpage. (You can open it locally with your browser, ignore the javascript that is used for appending code onto the page).
 - Take the code from [here](https://github.com/ChJus/HTML-and-CSS-Tutorial/tree/main/template).
